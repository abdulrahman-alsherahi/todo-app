# 📝 Python To-Do List CLI Application

A simple and interactive **command-line To-Do List application** built with Python.  
This project allows users to manage their daily tasks directly from the terminal.

---

## 🚀 Features

- ➕ Add new tasks
- 📋 View all tasks
- ❌ Delete tasks
- 🔁 Interactive menu system
- 🧠 Beginner-friendly Python logic

---
